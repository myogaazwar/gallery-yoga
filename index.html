About

Halo Nama Saya Muhammad Yoga Azwar, Saya Mahasiswa Semester 6 di Universitas Bina Sarana Informatika, Jurusan Sistem Informasi, Saya memiliki beberapa hobby yaitu bermain game, menonton film, dan bermain gitar, tetapi akhir-akhir ini saya mulai menghabiskan waktu saya mempelajari ulang HTML & CSS, dikarenakan pada semester sebelumnya saya merasa saya kurang mengetahui lebih dalam HTML & CSS.

Maka dari itu Website ini saya buat menggunakan framework css yaitu bootstrap, dengan mengikuti beberapa tutorial di youtube, dan saya juga menambahkan beberapa menu seperti lightbox menggunakan script js, yang menurut saya sangat membantu untuk memperindah website gallery ini. saya akui website gallery ini masih memiliki beberapa kekurangan, tapi untuk kedepannya saya akan terus belajar :) terimakasih sudah berkunjung... ^-^
